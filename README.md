# FREEDOM — 7 Pillars

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Latest Release](https://img.shields.io/github/v/release/dornori/freedom-7-pillars)](https://github.com/dornori/freedom-7-pillars/releases)

**FREEDOM** is a collaborative social manifesto that defines **7 foundational pillars** for a free and thriving humanity.

### The FREEDOM Acronym

| Letter | Pillar          | Core Idea                          |
|--------|-----------------|------------------------------------|
| **F**  | **Foundation**  | The bedrock principles of society  |
| **R**  | **Resilience**  | Strength to withstand challenges   |
| **E**  | **Exchange**    | Free and fair interaction          |
| **E**  | **Equilibrium** | Balance and harmony                |
| **D**  | **Defense**     | Protection of rights and freedoms  |
| **O**  | **Ownership**   | Right to control one's life & property |
| **M**  | **Mobility**    | Freedom of movement and choice     |

## 📖 About This Manifesto

This living document explores how these **7 pillars** can guide individuals, communities, and societies toward greater freedom and human flourishing. It is designed to be read, discussed, improved, and expanded through global open collaboration.

## How to Participate

- Read or download the latest version from the [Releases](https://github.com/dornori/freedom-7-pillars/releases) page
- Open an **Issue** to suggest improvements or discuss ideas
- Submit edits via **Pull Request** (preferred method)
- Browse or contribute to the [Wiki](https://github.com/dornori/freedom-7-pillars/wiki) for deeper guidelines

## 📂 Repository Contents

- `freedom-manifest.odt` — Main editable document (LibreOffice)
- `freedom-manifest.fodt` — Flat XML version (recommended for better Git diffs)
- `CHANGELOG.md` — Record of all changes

## 🛡️ Collaboration & Versioning

All changes to the manifesto go through **Pull Requests** on the protected `main` branch.  
Official versions are published as **GitHub Releases** with permanent tags (`v1.0.0`, `v1.1.0`, …).
