## 📂 Archives

The `archives/` folder contains historical snapshots of previous official versions of **The FREEDOM Framework**.

This folder serves as a permanent record so that any past version can be easily accessed, compared, or referenced.

### Structure
archives/
├── v1.0.0/
│   ├── freedom-framework-v1.0.0.odt
│   ├── freedom-framework-v1.0.0.fodt
│   └── freedom-framework-v1.0.0.pdf
├── v1.1.0/
│   └── ...
└── v2.0.0/

Each subfolder is named after the official version number (e.g. `v1.0.0`, `v1.1.0`, `v2.0.0`) and contains the complete set of files from that release.

### Purpose

- Preserve exact copies of every **official release**
- Allow users and researchers to access any previous version
- Maintain a stable archive independent of changes in the `manifest/` folder
- Support transparency and version history

Only versions that have been officially published via **GitHub Releases** are stored here.

You can browse the [`archives/`](https://github.com/dornori/freedom-7-pillars/tree/main/archives) folder to view all previous versions.
